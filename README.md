# Cici AI - Asisten Cerdas

Cici AI adalah aplikasi web berbasis chat yang berfungsi sebagai asisten cerdas. Dibangun menggunakan **HTML**, **Tailwind CSS**, dan **JavaScript**, Cici AI dapat membantu pengguna dengan berbagai pertanyaan, saran, atau tips cepat.

---

## Demo

> Masukkan URL demo jika tersedia, misal:  
> [Cici AI Demo](https://ciciai.netlify.app/)

---

## Fitur

- Tampilan modern dan responsif dengan **Tailwind CSS**
- Mode gelap dengan efek **glassmorphism** pada header
- Chat interaktif antara **user** dan **AI**
- Pesan cepat (Quick Actions) untuk memulai percakapan
- Animasi mengetik AI (Typing Indicator)
- Penyimpanan chat lokal menggunakan **LocalStorage**
- Unggah file sederhana (hanya menampilkan nama file)
- Menu samping untuk **Chat Baru** dan **Hapus Semua Chat**

---

## Teknologi

- [Tailwind CSS](https://tailwindcss.com)
- [Font Awesome](https://fontawesome.com) untuk ikon
- [Inter Font](https://fonts.google.com/specimen/Inter)
- JavaScript (Vanilla JS)
- API: [`https://api.sansekai.my.id/api/ai/cici`](https://api.sansekai.my.id/api/ai/cici)
